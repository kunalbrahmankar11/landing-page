# landing-page
TASK 01- Create a Landing Page
The internship involved creating a small landing page using HTML and CSS to brush up on front-end basics.
In this task, you will create a landing web page using basic HTML, And CSS. This is a 
very basic project for beginners. After completion of this project, you will be able 
to level up your CSS. In this project, you will learn how to divide a page into the header,
footer, body part, and griding system in web page.
